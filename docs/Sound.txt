Quickdoc for EMAPI-Sound
Documentation is shown with C-Syntax.
 
object Sound is an array representing {string id = "emapidef", string path = ""}

void GDT.resetMusic() : removes all background music from the game
void GDT.resetMusicByStage(int stage) : removes all background music from given stage
	-int stage : a given office/game stage. is between 1 and 4
void GDT.addMusic(Sound music = Sound(default))
	-Sound music : the file to play
	-NOTE: music.path must be an .ogg file
